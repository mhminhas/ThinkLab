Available Endpoints

Generate text content using AI models

Create images from text descriptions

Retrieve user projects and data

Analyze datasets with AI insights

Code Examples

// Generate text content with ThinkLab API
const response = await fetch('https://api.thinklab.ai/v1/generate/text', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: 'Write a blog post about AI in healthcare',
    max_tokens: 1000,
    model: 'thinklab-gpt-4'
  })
});

const data = await response.json();
console.log(data.content);
